# Sangraha LLC Website

A professional single-page React website for Sangraha LLC, a web development company specializing in AI-powered portfolio solutions for photographers and artists.

## About Sangraha LLC

Sangraha LLC creates stunning portfolio websites for photographers and artists, enhanced by cutting-edge AI technology. Our services include:

- **AI Image Pipeline**: Automated metadata management, location tagging, and intelligent photo organization
- **Smart Portfolio Grouping**: Organize portfolios by theme, style, location, or people
- **Content Enhancement**: AI-powered caption and hashtag generation for social media
- **Social Media Integration**: Automatic uploads to Instagram, Facebook, and other platforms

## Technologies

- React 19
- Vite
- Modern CSS with responsive design

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```
